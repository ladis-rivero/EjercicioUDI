# EjercicioUDI
Para la clase de UDI III; octavo ejercicio
-------------------------------------------------
Para la materia de Sara, creamos un sistema de administración de transacciones alrededor de la empresa/cliente/negocio Monky.
Con esto esperamos entregar un sistema que permita a los dueños de este poder administrar su trabajo de forma más eficiente, incluyendo tener un control acerca de las transacciones monetarias y los clientes que las realizan.
